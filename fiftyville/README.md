# Fiftyville

CS50 Week 7 - SQL

## What it is

A SQL detective mystery. Use SQL queries to solve a crime that took place in Fiftyville.

## The Mystery

The CS50 Duck has been stolen! The town of Fiftyville has called upon you to solve the mystery. Authorities believe the thief escaped by plane. Who took the duck? Where did they escape to?

## How it works

Write SQL queries in numbered files (1.sql, 2.sql, etc.) to:
- Analyze crime scene reports
- Check interview transcripts
- Review ATM transactions
- Examine phone records
- Search flight passenger lists
- Cross-reference all evidence

## Files

- `log.sql` - My investigation notes and queries
- `answers.txt` - Final solution (thief, accomplice, destination)
- `fiftyville.db` - SQLite database with all the evidence

## Usage
```bash
sqlite3 fiftyville.db < log.sql
```
