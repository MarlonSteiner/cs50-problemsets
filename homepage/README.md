# Homepage

CS50 Week 8 - HTML, CSS, JavaScript

## What it is

A personal website built with HTML, CSS, and JavaScript.

## Features

- Multiple pages with consistent navigation
- Responsive design
- Custom styling with CSS
- Interactive elements with JavaScript

## Technologies

- HTML5
- CSS3
- JavaScript
- Bootstrap (optional)

## Usage

Open `index.html` in a web browser.

Or run a local server:
```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Files

- `index.html` - Home page
- `styles.css` - Custom styles
- Other HTML pages as needed

## Requirements

- At least 4 different HTML pages
- At least one list, table, image, and link
- At least one stylesheet
- At least one use of Bootstrap
- At least one use of JavaScript
